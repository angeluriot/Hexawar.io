# Install

### 1. Dependecies

* **[NodeJS](https://nodejs.org/en/)**
* **[npm](https://www.npmjs.com)**

**Windows : [Download installer](https://nodejs.org/en/download/)**.

**Unix :**
```shell
$ sudo apt install nodejs
$ sudo apt install npm
```

<br/>

### 2. Setup

```shell
$ npm install express socket.io
$ npm install -D nodemon
$ npm run dev
```
