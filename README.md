# Install

### Dependecies

* **[NodeJS](https://nodejs.org/en/)**
* **[npm](https://www.npmjs.com)**

**Windows : [Download installer](https://nodejs.org/en/download/)**.

**Unix :**
```Shell
$ sudo apt install nodejs
$ sudo apt install npm
```

<br/>

### Setup

```Shell
$ npm install express socket.io
$ npm install -D nodemon
$ npm run dev
```
